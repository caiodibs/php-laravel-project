# PHP Laravel project

## Ferramentas usadas para o projeto

### Frontend
* 1. HTML5 (tags e WAI-ARIA)
* 2. CSS
* 3. Bootstrap 4.0 e dependências como jQuery 3.3.1 e popper 1.14.3

### Backend
* 1. PHP 7.2.10
* 2. MariaDB 10.3
* 3. Composer 1.7.2
* 4. Laravel 5.7
* 5. PHP MyAdmin 4.8

* . DB criado com o nome de laravel
